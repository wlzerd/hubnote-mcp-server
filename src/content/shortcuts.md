# 단축키와 팁

자주 쓰는 키보드 단축키와 작업 패턴을 정리합니다.

## 글로벌 단축키

| 키 | 동작 |
|---|---|
| Cmd / Ctrl + K | 검색 팔레트 열기 |
| Esc | 모달 / 팔레트 닫기 |

## 페이지 편집 (BlockNote)

| 키 | 동작 |
|---|---|
| `/` | 블록 메뉴 (헤딩 / 리스트 / 표 / 이미지 등) |
| `@` | 멘션 자동완성 |
| Cmd / Ctrl + B | 굵게 |
| Cmd / Ctrl + I | 기울임 |
| Cmd / Ctrl + U | 밑줄 |
| Cmd / Ctrl + K | (텍스트 선택 상태) 링크 추가 — 글로벌 검색과 컨텍스트로 분기 |
| Cmd / Ctrl + Z | 실행 취소 |
| Cmd / Ctrl + Shift + Z | 다시 실행 |
| Tab | 리스트 항목 들여쓰기 |
| Shift + Tab | 들여쓰기 해제 |
| Enter (빈 리스트 항목에서) | 리스트 종료 |

## HTML View 페이지

| 키 | 동작 |
|---|---|
| Cmd / Ctrl + E | 보기 ↔ 편집 모드 토글 |

편집기 안에서는 CodeMirror 의 표준 단축키가 작동합니다 — Cmd+F 검색, Cmd+/ 주석 토글 등.

## 모바일

- 좌측 가장자리에서 swipe 으로 사이드바 드로어 열기
- 워크스페이스 레일은 사이드바 안에 함께 슬라이드됩니다
- 데스크톱의 hover 메뉴 (페이지 ⋯ 등) 는 모바일에서 길게 누르기로 작동

## 자주 쓰는 패턴

### 페이지를 여러 사람이 동시에 보기

별도 설정 없이 같은 URL 을 공유하면 됩니다. 멤버라면 자동으로 협업 — 다른 사람의 커서가 실시간으로 보입니다.

비-멤버에게 보여주려면 페이지 메뉴 → "공유 링크" 발급 후 그 URL 을 전달.

### 같은 데이터를 여러 보드에 보이기

[데이터베이스](/help/database) 의 카드는 페이지 link 라, 한 페이지를 여러 보드에 동시에 등록할 수 있습니다.

- "프로젝트 보드" 와 "검토 캘린더" 양쪽에 같은 페이지를 카드로 추가
- 페이지 본체의 status / 마감일을 바꾸면 양쪽에 즉시 반영

### 비공개 작업 노트

- 개인 워크스페이스에서 자유롭게 작성
- 또는 팀 워크스페이스 안에서 페이지 가시성을 "비공개" 로 (본인만 보임)

### 운영 페이지 만들기

내부 도구 / 대시보드 / 폼은 [HTML View](/help/html-view) 페이지로. 워크스페이스 데이터 row 와 연동되는 자체 도구를 50KB JS 안에서 만들 수 있습니다.

### 정기 회의록

페이지 트리에 "회의록" 부모 페이지를 만들고 매주 자식 페이지를 새로 만드세요. 사이드바에서 펼치면 모든 회의록이 한눈에 보입니다.
