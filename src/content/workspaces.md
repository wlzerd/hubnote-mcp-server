# 워크스페이스

hubNote 의 모든 페이지는 워크스페이스 안에 있습니다. 워크스페이스는 사용자 자신만 보는 개인 공간일 수도 있고, 여러 멤버가 공유하는 팀 공간일 수도 있습니다.

## 개인 워크스페이스

가입 즉시 자동으로 1개 만들어집니다. 본인만 볼 수 있고, 멤버를 추가할 수 없습니다.

- 이름은 자유롭게 변경 가능
- 페이지의 "공유 / 비공개" 토글은 의미가 없습니다 (어차피 본인뿐)
- 다른 사람과 협업해야 하면 팀 워크스페이스를 따로 만들면 됩니다

## 팀 워크스페이스

화면 좌측의 워크스페이스 레일에서 + 버튼을 눌러 만듭니다.

생성 모달에서:

1. (선택) 이미지를 첨부합니다 — 5MB 이하 image/* 파일
2. 워크스페이스 이름을 입력합니다
3. "만들기" 버튼

성공하면 곧바로 초대 링크가 발급되고 모달의 다음 단계에서 복사 / 새 링크 발급 / 닫기 중 하나를 고를 수 있습니다.

만든 사람이 자동으로 그 워크스페이스의 **소유자(owner)** 가 되며, 모든 권한을 자동 통과합니다. 자세한 권한 모델은 [역할과 권한](/help/roles-permissions) 을 참조하세요.

## 멤버 초대

두 가지 방법이 있습니다.

### @username 직접 초대

워크스페이스 설정의 멤버 탭에서 상대방의 @아이디로 초대합니다. 상대방이 가입자라야 합니다.

### 초대 링크

워크스페이스 설정의 초대 링크 탭에서 발급. 링크를 받은 사람이 클릭하면 가입 후 자동으로 워크스페이스에 합류합니다.

- 한 워크스페이스에 활성 링크 하나
- "새 링크 발급" 으로 회전 — 노출이 의심되면 회전하세요
- 비활성화 가능

> 초대 링크가 외부에 유출되면 가입 절차만 거치면 누구나 들어옵니다. 외부 채널에 올렸다면 즉시 회전을 권장합니다.

## 워크스페이스 설정

좌측 워크스페이스 레일의 활성 워크스페이스 아이콘 위에서 우클릭 (또는 사이드바 상단의 톱니 아이콘).

- 이름 변경
- 이미지 변경 (5MB 이하)
- 삭제 (소유자만 가능, 모든 페이지가 같이 사라짐)

## 멤버 관리

워크스페이스 설정의 멤버 탭에서 가능한 작업:

- 멤버 목록 / 가입일 확인
- 역할 부여 / 변경
- 멤버 제거

소유자는 제거할 수 없습니다. 소유자 변경 기능은 아직 없습니다 — 필요하면 새 워크스페이스를 만들고 페이지를 옮기는 식으로 우회합니다.

제거된 멤버가 그 워크스페이스에서 만든 페이지는 그대로 남되, 작성자 정보는 비워집니다.

## 워크스페이스 전환

좌측 레일의 워크스페이스 아이콘을 클릭. 활성 워크스페이스는 좌측 막대로 표시되며 둥근 모서리가 사각으로 변합니다.

활성 워크스페이스는 브라우저에 저장되어 다음 접속 시에도 같은 곳에서 시작합니다. 만약 그 워크스페이스에서 제거되면 자동으로 개인 워크스페이스로 돌아갑니다.
