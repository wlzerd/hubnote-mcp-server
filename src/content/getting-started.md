# 시작하기

hubNote 가입과 첫 단계를 안내합니다.

## 로그인

Google OAuth 로 로그인합니다. `auth.discof.com` 으로 한 번 이동했다가 돌아옵니다.

처음 로그인하면 자동으로 다음이 일어납니다.

- 사용자 계정이 만들어집니다
- 개인 워크스페이스 1개가 자동 생성됩니다
- 약관 / 개인정보 처리방침 동의 화면이 한 번 뜹니다 (이후 약관 버전이 바뀌면 다시 뜸)

## 사용자명 (@handle)

자동 생성 형식: `user_xxxxxxxx` (예: `user_3f9a2b1c`).

계정 설정에서 직접 변경할 수 있습니다.

- 소문자 + 숫자 + `_`
- 3~30자
- 첫 글자는 알파벳

> 한 번 변경하면 30일 동안 다시 변경할 수 없습니다. 멘션이 깨지지 않도록 둔 의도적인 제약입니다.

## 프로필

표시 이름과 아바타는 계정 설정에서 변경합니다. 이메일은 OAuth 가 결정하므로 직접 바꿀 수 없습니다.

## 보안 (TOTP)

계정 설정의 "보안" 탭에서 TOTP MFA 를 등록할 수 있습니다 (Google Authenticator, 1Password 등).

등록 시 일회용 **복구 코드 8개** 를 받습니다. 인증기를 분실해도 이 코드 중 하나로 본인 인증할 수 있습니다.

- 복구 코드는 1회용입니다. 다 쓰면 새로 발급할 수 있습니다.
- 등록 직후 한 번만 화면에 표시되니 안전한 곳에 저장하세요.

## 계정 삭제와 복구

계정 설정에서 삭제를 요청하면 14일 동안 보류 상태가 됩니다.

- 14일 이내 가입 이메일로 받은 복구 링크를 클릭하면 그대로 복원됩니다
- 14일이 지나면 자동으로 영구 삭제됩니다
- 본인이 작성한 페이지와 파일은 그대로 남되, 작성자 정보는 비워집니다
- 보류 기간 중 로그인하면 복구 안내 페이지로 이동하고 일반 기능은 차단됩니다
