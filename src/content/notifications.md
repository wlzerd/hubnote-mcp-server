# 알림

hubNote 의 알림은 두 채널로 전달됩니다.

- **벨 아이콘** — 인앱, 우상단의 종 모양
- **이메일** — 가입 시 받은 Google 계정 주소로 발송

## 벨 아이콘

화면 우상단의 종 아이콘. 새 알림이 있으면 빨간 배지가 켜집니다.

- 30초마다 자동으로 폴링하여 갱신
- 클릭하면 드롭다운으로 최근 알림 목록
- 알림 항목을 클릭하면 해당 페이지로 이동
- "모두 읽음" 으로 한 번에 비울 수 있습니다

## 알림이 발생하는 상황

### 1. 멘션

페이지 본문에서 누군가 회원님을 `@사용자명` 으로 멘션하면:

- 즉시 벨 알림 + 이메일 발송
- 같은 멘션을 그 사람이 다시 저장해도 재발송되지 않습니다 (save-time diff)
- 본인이 본인을 멘션해도 알림은 가지 않습니다

### 2. 어드민 액션

시스템 어드민이 본인 계정에 다음 작업을 하면 자동 이메일이 옵니다.

| 액션 | 의미 |
|---|---|
| 관리자 부여 / 회수 | 어드민 권한 변경 |
| 계정 정지 / 해제 | 일시 정지 / 해제 |
| 사용자명 쿨다운 리셋 | 30일 제한 풀림 |
| 동의 bump | 약관 버전 강제 갱신 — 다음 로그인 때 ConsentGate 재노출 |

벨 아이콘에는 표시되지 않으며, 이메일로만 통지됩니다.

### 3. 환영 이메일 (가입 시)

첫 로그인 시 한 번만 발송됩니다.

- DB 의 `welcome_email_sent_at` 플래그로 멱등 처리 — 절대 중복 발송되지 않습니다

### 4. 계정 삭제 / 복구

- 계정 삭제 요청 시 — 14일 보류 안내 + 복구 링크 이메일
- 보류 기간 만료 14일 후 자동 영구 삭제 (별도 확인 메일 없음)
- 복구 링크 클릭 시 즉시 복원 — 별도 확인 메일 없음

## 이메일 설정

- 발신 도메인 — `discof.com` (Resend 검증된 도메인)
- 수신 주소는 가입 시 Google 에서 받은 이메일 — 직접 변경 불가
- 마케팅 / 광고 이메일은 절대 발송되지 않습니다 (가입 시 OAuth 동의 화면에 명시)
- 답장은 `support@discof.com` 으로 전송됩니다 (Cloudflare Email Routing 으로 운영자 메일함에 전달)

## 알림이 안 올 때 점검

이메일 미수신:

- 가입 이메일 주소를 다시 확인 (계정 설정에서 표시됨)
- 스팸 / 프로모션 폴더 확인
- 회사 메일 서버에서 외부 발신 차단 가능성 — `discof.com` 화이트리스트 요청

벨 알림 미반영:

- 30초 폴링 주기를 기다리거나 새로고침
- 다른 브라우저 탭에서는 즉시 안 뜨고 다음 폴링 때 뜸
- 워크스페이스에서 빠진 후 받은 멘션은 표시되지 않습니다

## 프라이버시

이메일은 위 4가지 목적 외에 사용되지 않습니다. 자세한 내용은 [개인정보처리방침](/privacy) 의 외부 처리 위탁 절을 참조하세요.
